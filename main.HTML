スプラトゥーン PCラグの世界へ

ラグを活用してゲームを極めよう。新しい技術とヒントで一歩先へ。

ラグを使った高所移動の例。素早いポジショニングが可能。

透明ラグで敵を翻弄。戦略的な動きを。

ホコを使ったラグで一気に攻め込む。

ソフトウェアの使い方

1. ソフトウェアのインストール

公式サイトからラグツールをダウンロードし、指示に従ってインストール。

2. 初期設定

ネットワーク設定を最適化し、ラグの種類に応じて調整。

3. Limitラグの設定

スライダーで帯域幅を制限し、通信速度を制御。

4. Blockラグの設定

特定のパケットをブロックして通信を一時遮断。

5. Queueラグの設定

パケットの優先順位を変更し、高度なラグを実現。

6. テスト環境の構築

カードのテスト環境を確認するオフラインモードで動作します。

7. ラグのタイミング調整

ゲーム内のアクションに合わせてラグを適用。

8. エラーのトラブルシューティング

ログを確認してエラーコードを解析。

9. ネットワークの安定化

ルーターを設定して安定したラグ環境を維持。

10. ラグの保存設定

プロファイルを保存して設定を再利用。

11. 複数ラグの組み合わせ

LimitとBlockを同時に適用。

12. ホコラグの設定

ホコモード専用のラグ設定を適用。

13. 透明ラグの設定

特定のパケットを操作して透明状態を保つ。

14. ラグの自動化

マクロやスクリプトでラグを自動化。

15. ラグのログ確認

実行ログを確認して動作を検証。

16. ゲーム内設定との連携

ゲームのコントローラー設定と同期。

17. ラグのバックアップ

設定をクラウドにバックアップ。

18. アップデートの確認

ツールの最新バージョンを確認。

19. コミュニティ設定の利用

他のユーザーの設定を参考に。

20. 安全なラグ運用

規約違反を避け、安全にラグを使用。

ラグの種類

1. Limit

通信速度を低下させることで発生するラグ。

最も使われることが多く、汎用性が高い。

2. Block

通信を遮断することで発生するラグ。

リミットと同様にほとんどのラグで使用できる。

3. Queue

キューを利用してより高度に制限することで発生するラグ。

主にに微ラグで使用される。

ラグの技術

スプラトゥーン2, 3

1. 新浮き

使用ラグ : Limit

難易度 : 中

高所への移動で使用される。様々な場面で活躍する。

2. ホコジャン

使用ラグ : Queue

難易度 : 中

ホコを持ってスーパージャンプを行う。浮き系のラグと組み合わせられる事が多い。

3. ホコジェッパ

使用ラグ : Queue

難易度 : 中

ソロでも高所の足場に乗れる。

スプラトゥーン3

4. 一回浮き

使用ラグ : Limit

難易度 : 最低

ラグ一回で浮くことが出来る最も簡単なラグ。

5. 旧浮き

使用ラグ : Block

難易度 : 高

高所に移動する時に使われるが、新浮きラグがあるため使用頻度は低め。

6. 旧浮き応用

使用ラグ : Limit + Block

難易度 : 最高

浮きラグ系の中で最も難しい。出来たら凄い。

7. ジャンプラグ

使用ラグ : Limit, Block

難易度 : 中

小さな段差や少し遠くの足場への移動で使用される。汎用性が高くよく利用される。

8. 落ち浮きラグ

使用ラグ : Block

難易度 : 中

高所への移動に使用されるが、新浮きがあるためやはり使用頻度は低め。

9. 落ち浮きラグ応用

使用ラグ : Block

難易度 : 最高

落ち浮きラグの応用。かなり難しい。

10. 透明ラグ

使用ラグ : Limit

難易度 : 低

透明になれる。ダメージは普通に食らう。非公開ラグ。

11. 浮上ラグ

使用ラグ : Block

難易度 : 最高

かなり高くまで上昇できる。運要素も大きくかなり難しい。

12. ホコショクワン

使用ラグ : Queue

難易度 : 中

ソロで高所の足場に乗れる。

13. 吹っ飛びラグ

使用ラグ : Queue

難易度 : 中

実用性は低いが見た目のインパクトが大きい。

スプラトゥーン2

14. 旧浮き

使用ラグ : Block

難易度 : 中

スプラ2では新浮きと同じくらい使用されている。

15. 旧浮き応用

使用ラグ : Limit + Block

難易度 : 最高

タイミングがかなり複雑な上に、スプラ2だと回線落ちのリスクも高い。

16. 浮きスラ

使用ラグ : Limit, Block

難易度 : 中

ラグ一回で様々な足場に乗れる便利なラグ。

17. 新浮きスラ

使用ラグ : Limit

難易度 : 中

こちらも浮きスラと同様に便利なラグ。

18. スライドラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 最低

最も簡単でシンプルなラグ。

19. スライドラグ(ビーコン無し、スライド有り)

使用ラグ : Limit, Block

難易度 : 低

ビーコンがない分少し難易度は上がるが、それでも簡単。

20. スライドラグ(ビーコン、スライド無し)

使用ラグ : Limit, Block

難易度 : 低

そもそも上２つのスライドラグで十分なため使用されることは少ない。

21. ジャンプラグ

使用ラグ : Limit, Block

難易度 : 中

遠くの足場や少しの段差で活躍する。簡単で汎用性も高い。

22. ジャンプラグ(落下)

使用ラグ : Limit, Block

難易度 : 中, 高

ジャンプラグの応用。場所によっては少し難しい。

23. 着地ラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 低

今ではジャンプラグの下位互換。

24. 着地ラグ(ビーコン無し)

使用ラグ : Limit, Block

難易度 : 低

ビーコンが必要ない分使いやすいが、それでもジャンプラグの下位互換。

25. 落ちラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 中

地面の下の足場に乗る時に使われる。場所によって難易度が変わる。

26. 落ちラグ(ビーコン無し)

使用ラグ : Limit, Block

難易度 : 中

ビーコンを使わない分、行ける場所は制限される。

27. ジェッパラグ

使用ラグ : Limit, Block

難易度 : 高

ジェッパラグで行けるところは全部、他のラグでもっと簡単に行くことが可能。

28. 前ラグ

使用ラグ : Block

難易度 : 中

ほとんど使われることはない。

29. 無敵ラグ

使用ラグ : Limit, Block

難易度 : 中

無敵+透明になれる。非公開ラグ。

30. 透明ラグ

使用ラグ : Limit, Block

難易度 : 最高

透明になれる。非常に回線が落ちやすい。

31. ホコ武器

使用ラグ : Queue

難易度 : 中

ホコと武器が合体する。実用性は無いが楽しい。

役立つヒント

不安定が出ないラグもあり、行動が制限されないため重宝される。
