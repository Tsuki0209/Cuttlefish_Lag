ラグの種類

1. Limit

通信速度を低下させることで発生するラグ。

最も使われることが多い、汎用性の高いラグ。

2. Block

通信を遮断することで発生するラグ。

主にスプラトゥーン2のラグや、3では一部のラグで使用されるラグ。

3. Queue

キューを利用してより高度に制限することで発生するラグ。

2、3ともに微ラグで使用されるラグ。

ラグの技術

共通

1. 新浮き

使用ラグ : Limit

難易度 : 中

直撃で大ダメージ。狭い通路で効果的。

2. ホコジャン

使用ラグ : Queue

難易度 : 中

ガチホコバトルでジャンプとラグを組み合わせ、素早く前進。

3. ホコジェッパ

使用ラグ : Queue

難易度 : 中

ガチホコとジェットパックをラグで強化。広範囲攻撃に。

スプラトゥーン3

4. 一回浮き

使用ラグ : Limit

難易度 : 最低

壁を登る時も有効。ジャンプと組み合わせて加速。

5. 旧浮き

使用ラグ : Block

難易度 : 高

インクが足りないと滑り落ちる。塗りを確認。

6. 旧浮き応用

使用ラグ : Limit + Block

難易度 : 最高

ボム投げ中に敵の反撃を避けるテクニック。

7. ジャンプラグ

使用ラグ : Limit, Block

難易度 : 中

スプラトゥーン3向け。高度なジャンプで敵を混乱させよう。

8. 落ち浮きラグ

使用ラグ : Block

難易度 : 中

落下中に浮きを組み合わせて移動。スプラトゥーン3で有効。

9. 落ち浮きラグ応用

使用ラグ : Block

難易度 : 高

高度な制御で敵を翻弄。精密なタイミングが必要。

10. 透明ラグ

使用ラグ : Limit

難易度 : 低

スプラトゥーン3向け。簡易的なラグで位置を隠す。

11. ショクワンラグ

使用ラグ : Limit, Block

難易度 : 高

ショクワンダー使用時にラグで動きを強化。攻撃力アップ。

12. ホコショクワン

使用ラグ : Queue

難易度 : 中

ガチホコとショクワンダーをラグで連携。突破力が増す。

13. 吹っ飛びラグ

使用ラグ : Queue

難易度 : 中

吹っ飛び中にラグで軌道を制御。意外な位置から反撃。

スプラトゥーン2

14. 旧浮き

使用ラグ : Block

難易度 : 中

ダメージ軽減効果あり。敵の攻撃を避けよう。

15. 旧浮き応用

使用ラグ : Limit + Block

難易度 : 最高

着地時に隙あり。ギアで着地時間を短縮。

16. 浮きスラ

使用ラグ : Limit, Block

難易度 : 中

予測移動が鍵。敵の癖を読もう。

17. 新浮きスラ

使用ラグ : Limit

難易度 : 中

素早い移動で敵を翻弄。シューターやブラスターで速度を活かそう。

18. スライドラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 最低

ビーコンを利用して素早く位置移動。味方のサポートに最適。

19. スライドラグ(ビーコン無し、スライド有り)

使用ラグ : Limit, Block

難易度 : 低

スライドで機動力を維持。狭いエリアで敵を出し抜く。

20. スライドラグ(ビーコン、スライド無し)

使用ラグ : Limit, Block

難易度 : 低

ビーコン活用で安全な位置へ移動。防御的な動きに有効。

21. ジャンプラグ

使用ラグ : Limit, Block

難易度 : 中

ジャンプ中にラグを活用し、敵の狙いを外す。タイミングが重要。

22. ジャンプラグ(落下)

使用ラグ : Limit, Block

難易度 : 中, 高

落下中にラグで位置をずらし、奇襲攻撃を仕掛ける。

23. 着地ラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 低

ビーコンへ着地時にラグで安全確保。素早い復帰に役立つ。

24. 着地ラグ(ビーコン無し)

使用ラグ : Limit, Block

難易度 : 低

着地時の隙をラグでカバー。敵の攻撃を回避。

25. 落ちラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 低

ビーコン利用で落下を制御。安全な位置へ移動。

26. 落ちラグ(ビーコン無し)

使用ラグ : Limit, Block

難易度 : 中

落下中にラグで敵の視線をずらし、奇襲を狙う。

27. ジェッパラグ

使用ラグ : Limit, Block

難易度 : 高

ジェットパック使用時にラグで位置をずらし、敵を混乱。

28. 前ラグ

使用ラグ : Block

難易度 : 中

前方移動中にラグで敵のエイムを外す。攻撃的な動きに。

29. 無敵ラグ

使用ラグ : Limit, Block

難易度 : 中

ラグで一時的な無敵状態を模倣。タイミングが鍵。

30. 透明ラグ

使用ラグ : Limit, Block

難易度 : 最高

ラグで敵から見えづらくなる。高度な制御で奇襲を。

31. ホコ武器

使用ラグ : Queue

難易度 : 中

ガチホコ持ち時に武器とラグを組み合わせ、攻防を強化。

役立つヒント

15. インク管理

インク残量をチェック。塗りすぎに注意。

ギア「インク効率アップ」で消費を軽減。

16. マップ確認

Xボタンでマップを開き、敵や塗り状況を確認。

定期的に見て、戦略を立てよう。

17. ギア選び

武器やプレイスタイルに合ったギアを装備。

例: スプラッシュボム多用なら「サブ効率アップ」。

18. シグナル活用

「カモン！」や「オッケー！」で味方と連携。

十字キーで簡単送信。状況を伝えよう。

19. ステージ理解

各ステージの地形やルートを把握。

高台や隠れスポットを覚えて有利に。

20. 練習モード

試し打ち場で武器や操作を練習。

エイムや移動を繰り返し、精度を上げよう。
