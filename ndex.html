ラグの種類

1. Limit

通信速度を低下させることで発生するラグ。

最も使われることが多く、汎用性が高い。

2. Block

通信を遮断することで発生するラグ。

リミットと同様にほとんどのラグで使用できる。

3. Queue

キューを利用してより高度に制限することで発生するラグ。

主にに微ラグで使用される。

ラグの技術

スプラトゥーン2, 3

1. 新浮き

使用ラグ : Limit

難易度 : 中

高所への移動で使用される。様々な場面で活躍する。

2. ホコジャン

使用ラグ : Queue

難易度 : 中

ホコを持ってスーパージャンプを行う。浮き系のラグと組み合わせられる事が多い。

3. ホコジェッパ

使用ラグ : Queue

難易度 : 中

ソロでも高所の足場に乗れる。

スプラトゥーン3

4. 一回浮き

使用ラグ : Limit

難易度 : 最低

ラグ一回で浮くことが出来る最も簡単なラグ。

5. 旧浮き

使用ラグ : Block

難易度 : 高

高所に移動する時に使われるが、新浮きラグがあるため使用頻度は低め。

6. 旧浮き応用

使用ラグ : Limit + Block

難易度 : 最高

浮きラグ系の中で最も難しい。出来たら凄い。

7. ジャンプラグ

使用ラグ : Limit, Block

難易度 : 中

小さな段差や少し遠くの足場への移動で使用される。汎用性が高くよく利用される。

8. 落ち浮きラグ

使用ラグ : Block

難易度 : 中

高所への移動に使用されるが、新浮きがあるためやはり使用頻度は低め。

9. 落ち浮きラグ応用

使用ラグ : Block

難易度 : 最高

落ち浮きラグの応用。かなり難しい。

10. 透明ラグ

使用ラグ : Limit

難易度 : 低

透明になれる。ダメージは普通に食らう。非公開ラグ。

11. 浮上ラグ

使用ラグ : Block

難易度 : 最高

かなり高くまで上昇できる。運要素も大きくかなり難しい。

12. ホコショクワン

使用ラグ : Queue

難易度 : 中

ソロで高所の足場に乗れる。

13. 吹っ飛びラグ

使用ラグ : Queue

難易度 : 中

実用性は低いが見た目のインパクトが大きい。

スプラトゥーン2

14. 旧浮き

使用ラグ : Block

難易度 : 中

スプラ2では新浮きと同じくらい使用されている。

15. 旧浮き応用

使用ラグ : Limit + Block

難易度 : 最高

タイミングがかなり複雑な上に、スプラ2だと回線落ちのリスクも高い。

16. 浮きスラ

使用ラグ : Limit, Block

難易度 : 中

ラグ一回で様々な足場に乗れる便利なラグ。

17. 新浮きスラ

使用ラグ : Limit

難易度 : 中

こちらも浮きスラと同様に便利なラグ。

18. スライドラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 最低

最も簡単でシンプルなラグ。

19. スライドラグ(ビーコン無し、スライド有り)

使用ラグ : Limit, Block

難易度 : 低

ビーコンがない分少し難易度は上がるが、それでも簡単。

20. スライドラグ(ビーコン、スライド無し)

使用ラグ : Limit, Block

難易度 : 低

そもそも上２つのスライドラグで十分なため使用されることは少ない。

21. ジャンプラグ

使用ラグ : Limit, Block

難易度 : 中

遠くの足場や少しの段差で活躍する。簡単で汎用性も高い。

22. ジャンプラグ(落下)

使用ラグ : Limit, Block

難易度 : 中, 高

ジャンプラグの応用。場所によっては少し難しい。

23. 着地ラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 低

今ではジャンプラグの下位互換。

24. 着地ラグ(ビーコン無し)

使用ラグ : Limit, Block

難易度 : 低

ビーコンが必要ない分使いやすいが、それでもジャンプラグの下位互換。

25. 落ちラグ(ビーコン有り)

使用ラグ : Limit, Block

難易度 : 中

地面の下の足場に乗る時に使われる。場所によって難易度が変わる。

26. 落ちラグ(ビーコン無し)

使用ラグ : Limit, Block

難易度 : 中

ビーコンを使わない分、行ける場所は制限される。

27. ジェッパラグ

使用ラグ : Limit, Block

難易度 : 高

ジェッパラグで行けるところは全部、他のラグでもっと簡単に行くことが可能。

28. 前ラグ

使用ラグ : Block

難易度 : 中

ほとんど使われることはない。

29. 無敵ラグ

使用ラグ : Limit, Block

難易度 : 中

無敵+透明になれる。非公開ラグ。

30. 透明ラグ

使用ラグ : Limit, Block

難易度 : 最高

透明になれる。非常に回線が落ちやすい。

31. ホコ武器

使用ラグ : Queue

難易度 : 中

ホコと武器が合体する。実用性は無いが楽しい。

役立つヒント

1. 不安定

不安定が表示されると動けなくなる。

不安定が出ないラグもあり、行動が制限されないため重宝される。
